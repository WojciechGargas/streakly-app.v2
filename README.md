# Streakly

Full-stack habit and task tracking application that helps users manage daily activities, build routines, and maintain streaks. Developed with ASP.NET Core, Angular, and PostgreSQL.

## Structure

- `apps/backend/Streakly.Api` - ASP.NET Core API
- `apps/backend/Streakly.Application` - application layer
- `apps/backend/Streakly.Core` - domain/core layer
- `apps/backend/Streakly.Infrastructure` - infrastructure layer
- `tests` - test projects
